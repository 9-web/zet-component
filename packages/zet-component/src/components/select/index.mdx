# Select (选择器)

## 基础展示
我们需要定义`treeData`属性，数据格式如下：
```
[
  {
    title: 'Node1',
    value: '0-0',
    key: '0-0',
    children: [
      { title: 'Child Node1', value: '0-0-1', key: '0-0-1' },
      { title: 'Child Node2', value: '0-0-2', key: '0-0-2' },
    ],
  },
  { title: 'Node2', value: '0-1', key: '0-1' },
];

```

## 基本
<Playground>
    <Select.TreeSelect treeData={treeData} />
</Playground>
