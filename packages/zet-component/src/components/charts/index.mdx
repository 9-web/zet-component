# Components.Chart (建模设计)
自动建模建模设计时候用此组件


## 基础展示

引入组件
```
import { Components } from 'zet-component';
```
####基础

<Playground style={{height:400}}>
  <State initial={{ show:'chart' }}>
    {({ state, setState }) => {
      const {Chart:{SliderChart}} = Components;
      return (
        <SliderChart
          data={data2}
          height={300}
          titles={[{alias:'CPU',key:'y'}]}
        ></SliderChart>
      )
    }}
  </State>
</Playground>
