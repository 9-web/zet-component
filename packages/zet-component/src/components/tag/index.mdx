# Tag (选择器)

## 基础展示
我们需要定义`data`属性，数据格式如下：
```
[
  {
    title: 'Node1',
    value: '0-0',
    key: '0-0',
  },
  { title: 'Node2', value: '0-1', key: '0-1' },
];

```
