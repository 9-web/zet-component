# TimeSelect

## 使用
`TimeSelect`用来在选择时间范围，最终返回秒，可以结合Form表单使用, 现阶段支持秒，分钟，小时

## data 配置
```
const data = {
  seconds: {
    key: 'seconds',
    selectData: [1, 5, 10, 20, 30],
    displayUnit: '秒',
  },
  minutes: {
    key: 'minutes',
    selectData: [1, 5, 10, 20, 30],
    displayUnit: '分钟',
  },
  hours: {
    key: 'hours',
    selectData: [1, 2, 4, 8, 12, 18],
    displayUnit: '小时',
  },
};

```
